# オリジナルECサイト


## 簡単な説明
アプリケーションのコンセプトは、「幅広い年齢の女性をターゲットに、様々な紅茶の種類から自分の好みの紅茶が購入できるオンラインショップサイト」を目指しました。
言語としてはPHPを使用し、依存関係をわかりやすくするためにMVCモデルで制作しました。
開発環境は、dockerコンテナを用いて構築しました。
